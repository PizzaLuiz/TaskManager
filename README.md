# TaskManager

Projeto para gerenciar tarefas,  aplicação permite que você cria tarefas, complete elas, e exclua.

## Rotas

- Salvar Task: POST http://localhost:8080/task
- Buscar todas as Tasks: GET http://localhost:8080/task
